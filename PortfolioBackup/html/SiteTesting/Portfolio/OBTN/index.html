Welcome to Overwatch by the Numbers

Our intention is to give overwatch players a better understanding of the game as well as their own personal play relative to the overall playerbase. This is done by gathering game data from all platforms that is analyzed using machine learning to form probabilistic trends. We use these trends as the basis for all data we provide and more information about our algorithm can be found under our "About" section.
